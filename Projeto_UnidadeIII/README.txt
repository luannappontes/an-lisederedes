Building a network of retweets using the Twitter API
Retweets network on Data Science topic.

Esta postagem faz parte da trajetória de aprendizado na disciplina de Análise de Redes 
(IMD1155 - T01–2021.1) da Universidade Federal do Rio Grande do Norte. 
Anteriormente criei uma rede com base nas páginas do Wikipedia, agora vamos utilizar a API do Twitter 
para construção de uma nova rede utilizando a linguagem Python e o Gephi.

Para o exercício é utilizado o tema Data Science.
